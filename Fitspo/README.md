#  Fitspo (Fashion Inspo) (IOS)

A native iOS mobile fashion inspiration application built on SwiftUI.

## Tech Stack
- Swift
- SwiftUI
- MVVM architecutre
- GitHub for version control

## Features
- Outfit inspiration feed
- Modular outfit cards
- Scalable architecture for recommendations

## Roadmap
- Personalization
- Visual similarity
- Frontend integration
- Backend integration
